**Explications** :
- Le créateur démarre un live avec sa caméra (LiveCamera.jsx)
- Le flux vidéo passe par Agora SDK (agoraClient.js)
- Agora gère la diffusion en temps réel vers tous les spectateurs
- Les spectateurs regardent le live via LivePlayer.jsx
- Les réactions (ReactionBar.jsx) sont envoyées en temps réel (via Supabase ou WebSocket)

Cette architecture garantit :
- Latence très faible
- Qualité vidéo stable
- Scalabilité massive
